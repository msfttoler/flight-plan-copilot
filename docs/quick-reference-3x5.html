Pro Tips
  • Open relevant files — Copilot reads your tabs for context
  • Name things well — getFlightPrice > gfp
  • Comment first, let Copilot write the code
  • Iterate — start rough, refine with follow-ups
  • Review always — you're the editor, not the typist
  • Use #file to attach specific files as context
  • Highlight code before asking inline chat
When to Use What
Know what you want? → Ghost Text
Transform code? → Inline Chat
Think out loud? → Chat Panel
Tests / docs / fix? → Slash Cmds
Project context? → @workspace
