💡 Tips That Make a Difference
  • Open relevant files before asking — Copilot reads your open tabs
  • Name functions well — calculateShippingCost beats calc
  • Write the comment first, then let Copilot write the code
  • Iterate in conversation — start rough, refine with follow-ups
  • Always review — Copilot is a first draft, you are the editor
🧭 When to Use What
Know roughly what you want? → Ghost Text (just type)
Have code to transform? → Inline Chat (⌘ I)
Need to think out loud? → Chat Panel (⌘ ⇧ I)
Want tests, docs, or fixes? → Slash Commands (/tests /doc /fix)
Need project-wide context? → @workspace in Chat
